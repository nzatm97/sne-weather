# Southern New England Weather

A static, multi-page weather site for GitHub Pages with a dark, modern dashboard on Home.

## Features

- **Branding:** Southern New England Weather
- **Multi-page layout:** Home, About Us, Contact, Social Media with persistent left sidebar nav
- **Home dashboard:**
  - Location search + recent searches
  - Radar animation timeline
  - NWS alerts list + polygons
  - Map layer toggles: radar, temperature overlay, wind overlay
  - Premium-style meteogram (temperature, precip probability, wind)
  - 5-day snapshot with colorful emoji weather icons
- **Placeholder slots:** hero photo feature and ad slot for future monetization
